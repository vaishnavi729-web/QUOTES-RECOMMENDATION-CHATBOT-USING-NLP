# QUOTES-RECOMMENDATION-CHATBOT-USING-NLP
Quotes Recommendation Chatbot using Rasa NLU delivers personalized quotes based on user emotions and preferences.
It supports categories like Motivation, Inspiration, Success, Love, and Humor.
Built with Rasa NLU and web integration via REST API, it provides a context-aware, interactive experience.
The chatbot encourages positivity, engagement, and emotional well-being.
